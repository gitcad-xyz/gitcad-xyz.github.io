MCP tool reference

The primary interface. Generated from the live registry — signatures and docs come from the code itself.

assembly_validate

assembly_validate(assembly_body: 'dict[str, Any]', parts: 'list[str]') -> 'dict[str, Any]'

Validate an assembly body (instances + mates) against its parts' interfaces: port-type compatibility and positional coincidence — the cross-domain co-design check (ADR-0008).

board_export_fab

board_export_fab(board: 'str', outdir: 'str') -> 'dict[str, Any]'

Validate and write the full fabrication package: Gerber X2 layers, Excellon drill, pick-and-place CSV, manifest.

board_validate

board_validate(board: 'str') -> 'dict[str, Any]'

Fab-readiness checks on a board document. Machine-readable.

feature_add

feature_add(model: 'str', op: 'str', params: 'dict[str, Any] | None' = None, inputs: 'list[str] | None' = None) -> 'dict[str, Any]'

Append an intent-level feature. Returns the new model text and the stable id assigned to the feature (never an ordinal index).

model_drawing

model_drawing(model: 'str', path: 'str', title: 'str' = 'part', sheet: 'str' = 'A3') -> 'dict[str, Any]'

Build the model and emit a dimensioned 2D drawing (SVG or PDF by file extension) of the final feature — front/top/right/iso, third angle.

model_export

model_export(model: 'str', path: 'str', fmt: 'str' = 'step') -> 'dict[str, Any]'

Build the model and export the final feature's shape to STEP or STL — the mechanical manufacturing deliverable.

model_measure

model_measure(model: 'str') -> 'dict[str, Any]'

Build against the best available kernel and return mass properties per feature — the deterministic oracle an agent verifies against.

model_new

model_new() -> 'dict[str, Any]'

Create an empty model; returns its canonical text form.

model_validate

model_validate(model: 'str') -> 'dict[str, Any]'

Build and run geometric validity checks per feature (watertight, self-intersection, ...). Machine-readable so the agent can act on it.

part_check_release

part_check_release(old_part: 'str', new_part: 'str') -> 'dict[str, Any]'

Interface-semver release gate (ADR-0009): given old and new part.json texts, classify the interface change and verify the version bump suffices. The check that stops shipping a breaking change as a patch.
